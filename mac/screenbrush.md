# SCREENBRUSH

Ferramentinha equivalente ao "zoom-it" do windows, pra destacar coisas na tela com linhas, setas e spotlight. Bom pra apresentações.

| Shortcut | Description |
| ----------- | ----------- |
| option + tab | Inicia e fecha o modo de apresentação |
| B | Brush |
| Q | Quick Arrow |
| SHIFT | Flashlight |
